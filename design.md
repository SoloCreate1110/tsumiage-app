# 積み上げアプリ - デザイン計画

## アプリ概要
時間や回数を「積み上げ」て記録し、成長を可視化してモチベーションを高めるアプリ。

## スクリーン一覧

### 1. ホーム画面 (Home)
- 登録済みの積み上げ項目一覧を表示
- 各項目の累計時間/回数をカード形式で表示
- 新規項目追加ボタン（FAB）

### 2. 項目詳細画面 (ItemDetail)
- 選択した項目の詳細情報
- タイマー開始/停止ボタン（時間タイプの場合）
- カウンター増減ボタン（回数タイプの場合）
- 今日/今週/今月/累計の統計表示

### 3. 統計画面 (Stats)
- 全項目の成長グラフ
- 日別/週別/月別の推移
- 達成バッジや記録表示

### 4. 設定画面 (Settings)
- アプリ設定（通知など）
- データエクスポート
- アプリ情報

## 主要コンテンツと機能

### ホーム画面
- **積み上げ項目カード**: アイコン、項目名、累計値、最終更新日
- **クイックアクション**: カードタップで詳細画面へ
- **新規追加FAB**: 右下に配置

### 項目詳細画面
- **タイマーモード**: 大きな円形タイマー、開始/停止/リセットボタン
- **カウンターモード**: 大きな数字表示、+/-ボタン
- **履歴リスト**: 直近の記録一覧
- **ミニ統計**: 今日/今週/累計の簡易表示

### 統計画面
- **グラフ**: 棒グラフで日別推移を表示
- **サマリーカード**: 総積み上げ時間/回数
- **達成記録**: 連続日数、最高記録など

## キーユーザーフロー

### フロー1: 時間を積み上げる
1. ホーム画面で項目カードをタップ
2. 詳細画面で「開始」ボタンをタップ
3. タイマーが動き始める
4. 作業終了時に「停止」ボタンをタップ
5. 時間が自動保存され、累計に加算

### フロー2: 回数を積み上げる
1. ホーム画面で項目カードをタップ
2. 詳細画面で「+」ボタンをタップ
3. カウントが1増加し、自動保存

### フロー3: 新規項目を追加
1. ホーム画面右下のFABをタップ
2. モーダルで項目名、タイプ（時間/回数）、アイコンを選択
3. 「追加」ボタンで保存
4. ホーム画面に新しいカードが表示

### フロー4: 成長を確認
1. 下部タブの「統計」をタップ
2. グラフで日々の積み上げ推移を確認
3. 累計時間/回数で成長を実感

## データモデル

### StackItem（積み上げ項目）
```typescript
interface StackItem {
  id: string;
  name: string;
  type: 'time' | 'count';
  icon: string;
  color: string;
  totalValue: number; // 秒 or 回数
  createdAt: string;
  updatedAt: string;
}
```

### StackRecord（積み上げ記録）
```typescript
interface StackRecord {
  id: string;
  itemId: string;
  value: number; // 秒 or 回数
  date: string; // YYYY-MM-DD
  createdAt: string;
}
```

## アニメーション

- **カウント増加**: スケールバウンス
- **カード追加**: フェードイン + スライドアップ
